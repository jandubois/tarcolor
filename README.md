# tarcolor
Perl program that can color output of `tar tvf` similarly to the way ls would

By Marc Abramowitz [http://marc-abramowitz.com](http://marc-abramowitz.com)

## Installation

<pre>
$ cp tarcolor.pl ~/bin/tarcolor
</pre>
